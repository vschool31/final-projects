Чтобы вставить картинку в свой сайт вам потребуется тег <img>. Но одного тега будет недостаточно. Вам нужно указать путь к картинке. Для этого используется атрибут src.
<img src = "путь к картинке">
Свою картинку вы можете превратить в карту. Это будет таже самая картинка, но в зависимости от места клика, будут будут активироваться разные ссылки. Для этого тегу <img> нужно указать атрибут usemap. Нужно придумать имя карты и написать в этот атрибут (обязательно с # перед именем).
<img src = "путь к картинке" usemap = "#имя">
Далее тегом <map> вы создаете карту. Чтобы связать карту с изображением, в теги map нужно написать атрибут name (значение должно совпадать с атрибутом usemap тега <img>, но уже без #).
<map name = "имя"></map>
На карту надо нанести зоны. Осуществляется это тегами <area>. Каждый тег обозначает одну зону на карте. Настройка зоны происходит слудующими атрибутами.
Атрибут shape указывает форму зоны. У него есть 3 значения: circle (форма в виде окружности), poly (в виде многоугольника), rect (в виде прямоугольника).
Атрибут coords определяет координаты точек зоны. Координаты точки обозначаются двумя числами: первое координата по оси X, вторая координата по оси Y. Численное значение координат представляет из себя количество пикселей, которое нужно отступить (по оси X и Y соответственно). Перечисляются координаты через запятую. Количество координат зависит от фигуры: circle - 3 (две координаты центра и радиус), rect - 4 (координаты верхнего левого угла и нижнего правого), poly - в зависимости от вида многоугольника (координаты для каждой вершины).
Атрибут href указывает ссылку на объект
<map name = "имя">
<area shape = "circle" coords= "X, Y, R" href = "ссылка №1"> <!--Круглая зоны-->
<area shape = "rect" coords = "X1, Y1, X2, Y2" href = "ссылка №2"> <!--Прямоугольная зоны -->
<area shape = "poly" coords = "X1, Y1, X2, Y2, X3, Y3" href = "ссылка №3"> <!--Треугольная зона-->
</map>
