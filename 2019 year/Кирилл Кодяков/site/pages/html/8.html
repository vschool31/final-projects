Формы предназначены для обмена данными между пользователем и сервером. Любые ваши действия на сайте обрабатываются сервером. Например, открытие сайта, переходы между страницами, ввод какой-либо информации. Если для первых двух пунктов за вас уже всё сделано (вам не нужно прописывать дополнительный код, чтобы ваш сайт можно было открыть), то для ввода информации и отправки её на сервер нужно использовать формы. Установка формы осуществляется тегом <form> с атрибутом action. В action указывается URL программы, которой передаётся управление данными (то есть, всё, что вы укажите между тегами <form> </form>, отправится в программу, указанную в атрибуте action).
<form action = "программа">
 ...
</form>
Для сбора информации с пользователя используется тег <input>. Без атрибутов этот тег представляет собой текстовое поле.
<input>

Однако, с помощью атрибута type, вид этого тега можно изменить.
Значение checkbox устанавливает вид флажка. Флажки позволяют выбрать из списка более одного варианта. Чтобы флажки были из одной группы, каждому <input> нужно указать атрибут name (он должен совпадать у каждого <input>).
<input type = "checkbox" name = "ar"> Первый элемент
<input type = "checkbox" name = "ar"> Второй элемент
<input type = "checkbox" name = "ar"> Третий элемент

Значение radio устаналивает вид переключателя. Из всех переключатель в списке одновременно может быть выбран только один. Как и в случае с checkbox, чтобы все переключатели относились к одной группе, у всех должны быть одинаковые атрибуты name.
<input type = "radio" name = "ar"> Первый элемент
<input type = "radio" name = "ar"> Второй элемент
<input type = "radio" name = "ar"> Третий элемент

Значение file превратит <input> в кнопку, по которой мы сможем выбрать файл.
<input type = "file">

Значение button превращает тег в кнопку. Но стоит отметить, что для этого имеется отдельный тег <button> (главное отличие только в том, что для <button> отображаемое имя кнопки вписывается между открывающим и закрывающим тегами, а для <input> в атрибут value).
<input type= "button" value = "Кнопка">

Но обмен данными между пользователем и сервером происходит не только с помощью <input>. В HTML имеется тег <textarea>, который представляет из себя область для ввода многострочного текста. От обычного <input> он отличается тем, что в нем допустимы перенос строк (которые сохраняются при отправке).
<textarea>Сюда можно вписать многострочный текст</textarea>
