В этом уроке будут рассмотрены основные операторы языка JavaScript. Для работы с переменными (точнее с их значениями), JavaScript поддерживает все стандартные операторы, известные со школы (сложение, умножение, вычитание, деление и прочие). Рассмотрим операторы, о которых вы могли не знать, а также некоторые особенности операторов в языке JavaScript.
Но для начала введем несколько терминов, касающиеся операторов (эти термины используются и в других языках программирования). Операнд - то, к чему применяется оператор. Например: 5 * 2 – оператор умножения с левым и правым операндами. Унарным называется оператор, который применяется к одному операнду. Например, оператор унарный минус меняет знак числа на противоположный. Бинарным называется оператор, который применяется к двум операндам.
С помощью бинарного плюса можно складывать не только чиса, но и строки. В этом случае две и более строк будут преобразованы в одну.
var a = "что-то" + "написали";
По итогу, в переменной a будет находиться строка "что-тонаписали". Если хотя бы один аргумент является строкой, то второй будет также преобразован к строке. Причем не важно, справа или слева находится операнд-строка, в любом случае нестроковый аргумент будет преобразован. Это приведение к строке – особенность исключительно бинарного плюса.
var a = "1"+ 2; // "12"
Унарный плюс преобразует переменную к числовому типу.
var a = "10"; // строка
+a; // число
Чтобы взять остаток от деления числа на число используется оператор %. Например, результатом выражения 5%2 будет число 1. Важной операцией любого языка программирование является увеличение или уменьшение переменной на единицу. Для этого существуют специальные операторы.
i++ // увеличение переменной на 1
i-- // уменьшение переменной на 1
