К РАЗДЕЛАМ
Иногда, в зависимости от условия, нужно выполнить различные действия. Для этого используется условный оператор if. Структура условия такова:
if(условие){
 команды
}
После if следуют круглые скобки, в которых содержится какое-то условие. Если это условие истинно, выполняется код, написанный в фигурных скобках. Если условие не выполняется, код в фигурных скобках пропускается. Если же нам нужно, чтобы что-то происходило в случае невыполнения условия, мы должны добавить необязательный блок else.
if(условие){
 команды
}
else{
 команды
}
Кроме того в JavaScript имеется конструкция else if, с помощью которой можно осуществлять несколько проверок и вполнять определенный код при определенном условии.
if(условие-1){
 команды
}
else if(условие-2){
 команды
}
else{
 команды
}
В JavaScript имеются специальные логические операторы || ('ИЛИ') и &&('И'), с помощью которых в if можно указывать несколько условий. Первый оператор используется, когда нас интересует одно из условий, а второй, когда нам нужно выполнение всех условий.
if(условие-1 || условие-2){...} // истинно, если выполняется хотя бы одно условие
if(условие-1 && условие-2){...} // истинно, если выполняются оба условия
Какие условия могут находиться в if? Например, сравнение каких-либо переменных.
if (a > b){...} // если a больше b
if (a < b){...} // если a меньше b
if (a >= b){...} // если a больше или равно b
if (a <= b){...} // если a меньше или равно b
Можно проверить равны ли переменные между собой. Для этого используются операторы == (равно), === (строгое равно), != (не равно). Если нас интересует только равенство значений переменных, то используется обычное равно. Если же нам важно, чтобы помимо значений переменых у них были одинаковые типы, используется строгое равно. Восклицательный знак обозначает частицу "не". Таким образом конструкция != обозначает "не равно".
if(a == b){...} // если a равно b
if(a === b){...} // если a равно b и при этом у них одинаковый тип данных
if(a != b){...} // если a не равно b
