Массив - упорядоченная структура данных, состоящая из элементов одинакового типа данных. Каждый элемент в массиве имеет свой индекс. Любой массив можно сравнить со списком, в котором рядом с каждым пунктом что-то указано (например, список продуктов). Объявление массива:
var имя_массива = [];
В квадратных скобках указываются элементы массива.
var mas = [1, 10, 100]; // массив, состоящий из цифр
Элементы массива нумеруются с нуля. Чтобы получить нужный элемент из массива – указывается его номер в квадратных скобках.
var mas = [1, 10, 100];
mas[0]; // 1
mas[1]; // 10
mas[2]; // 100
Можно изменять элементы массива и добавлять новые элементы:
mas[0] = 2; // изменили первый элемент
mas[3] = 1000; // добавили новый элемент
Длина массива узнается с помощью свойства length. Вызывается это свойства следующим образом:
mas.length;
Массив mas, используемый в примерах выше, называется одномерный. Помимо одномерных массивов существует многомерные массивы (массивы, состоящие из массивов). Самые популярные из многомерных массивов - двумерные (матрицы).
var matrix = [
 [1,2,3],
 [4,5,6],
 [7,8,9]
];
Если вызов элементы в одномерном массиве происходит с помощью указание его номера в квадратных скобках, то в двумерном массиве нам нужно два номера. Первый указывает номер массива (строка), второй - номер самого элемента (столбец).
alert(matrix(1,1)) // центральный элемент - 5
