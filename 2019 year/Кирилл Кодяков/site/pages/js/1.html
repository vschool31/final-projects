JsvaScript - язык программирования, который создавался с целью добавления сайтам динамичности. Он подключается напрямую к HTML с помощью тега </script>.
<script src="путь к файлу"></script>
Почему для создания сайтов используют именно этот язык программирования? Во-первых, с его помощью можно тесно взаимодействовать с HTML и CSS. Это позволяет осуществлять вам любые идеи по оформлению страниц, которые приходят вам в голову. Во-вторых, простые вещи в нем делаются просто. Вам не нужно писать кучу строк кода, чтобы, например, сделать объект на странице кликабельным. В-третьих, интерпретатор JavaScript присутствует во всех современных браузерах. Всё это делает его идеальным языком программирования для "оживления" сайтов.
В любой JavaScript-программе вы будете работать с информацией. Для хранения этой информации используются переменные. Переменная состоит из имени и выделенной области памяти, которая ему соответствут. Для объявления (создания) переменной используют ключевое слово var.
var *имя переменной*;
После объявления (или во время объявления) в переменную можно записать данные:
var message;
message = 'Hello, World!';
var number = 123;
В JavaScript не объявляют тип переменной (он определяется автоматически). В дальнейшем с этими данными можно будет работать, обращаясь к ним по имени переменной. Можно объявлять несколько переменных сразу:
var user = 'Igor', age = 25, message = 'Hello';
На имя переменной в JavaScript наложены некоторые ограничения: имя может состоять из букв, цифр (цифра не может быть первым символом имени) и двух символов ($ и _). Регистр также имеет значение: переменные num и Num - две разные переменные.
