К РАЗДЕЛАМ
У любых границ есть три свойства: цвет (border-color), ширина (border-width) и стиль (border-style). Для отображения границ должны быть заданы все 3 свойства. Цвет и ширина задаются любым известным способом. В border-style можно передать значения solid (сплошная), dotted (точечная), dashed (пунктирная), double (двойная), groove (объемная рифленая вдавленная), ridge (объемная рефленая выпуклая), inset (объемная вдавленная) или outset (объемная выпуклая).
div{
 border-color: #ffffff;
 border-width: 12px;
 border-style: dashed;
}
Чтобы каждый раз не прописывать все три свойства можно использовать свойство border, в которое передаются значения border-color, border-width и border-style в любой последовательности без разделителей.
div{
 border: dashed #ffffff 12px;
}
Помимо этих трех свойств существуют и другие. Например, свойство border-radius позволяет скруглить углы элемента. В это свойство передается значение (в пикселях, процентах и т.д.), на которое надо скруглить углы (чем больше значение, тем больше скругление). Задавать border-radius можно несколькими способами: передавая одно значение или несколько. В первом случае все углы скруглятся одинаково, а во втором - в зависимости от количества переданных значений (минимум одно, максимум четыре).
div{
 border-radius: 10%;
}
Необязательно задавать все границы элемента. В CSS существует возможность отображать отдельные границы элементы. Например, запись border-top-width отвечает за ширину верхней границы. По аналогии с этой записью можно обращаться ко всем четырем границам и их свойствам.
div{
 border-bottom: 13px solid red;
}
Перейдем к теням и прозрачности. За прозрачность элементов отвечает свойство opacity, которое принимает значения в диапазоне от 0.0 (полностью прозрачный) до 1.0 (полностью непрозрачный). Значения между этими границами соответствуют разным уровням прозрачности.
div{
 opacity: 0.5; //полупрозрачный элемент
}
Тень можно добавить либо к элементу (box-shadow), либо к тексту (text-shadow). В box-shadow можно передать от 2 до 6 параметров. У первых двух параметров должны быть числовые значения. Они будут указывать смещение тени от элемента по координатам x (первый параметр) и y (второй параметр). Положительные значения сместят элемент вправо и вниз, отрицательные значения сместят элемент влево и вверх.
div{
 box-shadow: 10px -10px; // Смещение тени на 10px вправо и 10px вверх
}
Третий и четвертый параметры также принимают числовые значения. Третий параметр отвечает за размытость тени. Чем больше значение этого парметра, тем сильнее размытость - тень становится больше и светлее. Отрицательные значения не допускаются.
div{
 box-shadow: 10px -10px 20px;
}
Четвертый параметр отвечает за размер тени: отрицательные значения уменьшают размеры со всех сторон, нулевое значение оставляет тень без изменений (тень будет того же размера, что и элемента), положительные значения увеличивают тень со всех сторон.
div{
 box-shadow: 10px -10px 20px 35px;
}
Пятый параметр отвечает за цвет тени. Указывать цвет можно любым удобным способом. Шестой параметр принимает только значение inset. Если он есть, то тень будет располагаться внутри элемента.
div{
 box-shadow: 10px -10px 20px 35px black inset;
}
Первые два параметры обязательны для этого свойства. Остальные 4 - нет. Стоит отметить одну вещь: если вы хотите указать цвет или сделать тень внутренней, вам необязательно для этого указывать третий и четвертый параметры. Например, можно просто указать координаты смещения и цвет.
div{
 box-shadow: 10px -10px black;
}
Как уже было сказано, text-shadow отвечает за тень текста. У него 4 параметра: два обязательных(смещение по x и смещение по y) и два необязательных (размытость и цвет). Работает всё так же, как и у box-shadow.
div{
 text-shadow: 13px -12px 34px white;
}
