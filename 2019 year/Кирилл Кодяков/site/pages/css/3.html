Правильно расположить элементы на своем сайте - это целая наука. В CSS существует огромное количество стилей, отвечающих за позиционирование, с помощью которых вы сможете расположить нужный вам элемент так, как вы хотите.
Прежде чем указать положение элемента, вам нужно определить метод позиционирование элемента относительно других элементов. Оcуществляется это с помощью свойства position со значениями absolute, fixed, relative, static или inherit.
При значении absolute элемент становится абсолютно позиционированным. Это значит, что данный элемент отображается на странице так, будто он на ней единственный. В свою очередь все остальные элементы отображаются так, будто абсолютно позиционированного элемента нет. С помощью этого можно располагать объекты на странице независимо друг от друга.
При значении fixed элемент ведет себя так же, как и со значением absolute, за исключением того, что он привязывается к определенному ему месту. При прокрутке странице он никуда не переместится.
При значении relative объект будет смещаться относительно своего первоначального расположения.
При значении static элемент располагается как обычно (точно так же, если бы мы не прописывали свойство position с этим значением).
Элемент со значением inherit наследует значение своего родительского элемента.
div{
 position: absolute;
}
После определения метода позиционирования можно смещать элемент с помощью свойств top, left, right, bottom. Численное значение показывает на сколько нужно сместить объект от выбранной границы.
div{
 position: absolute;
 top: 20px; /*смещение всех div на 20 пикселей сверху*/
}
Но элементы можно перемещать не только по осям X и Y. Вам может понадобится перекрывать одни элементы другими. В таком случае у вас должна быть возможность изменять порядок отображение этих элементов. Для этого существует свойство z-index. Чем больше его значение, тем "выше" лежит объект.
#elem{
 z-index: 1;
}
#obj{
 z-index: 2; /*Этот объект отобразится первым*/
}
