Трансформация - это какие-либо преобразования элемента. За трансформацию отвечает свойство transform. Это свойство применяет 2D или 3D преобразование к элементу. Оно позволяет смещать, вращать, масштабировать, наклонять любой элемент на странице.
В качестве значений свойства transform выступает одна или несколько функций (разделяемых пробелами), которые будут применяться в том порядке, в котором они определены.
Функция translate позволяет смещать элементы. Принимает два числовых значения: смещение по оси X и смещение по оси Y. Ведет себя подобно смещению элемента со свойством position: relative. Важная особенность функции translate в том, что при передаче в неё процентных значений проценты будут рассчитываться от размеров элемента.
div{
 transform: translate(50%, 50%);
}

/* Смещение элемента на половину своей высоты и ширины вниз и вправо */
Функция scale масштабирует элемент горизонтально и вертикально. Принимает два числовых значения без единиц измерения. Например, scale(2,2) увеличит размеры элемента в два раза. Можно передать только одно значение(оно будет использоваться сразу и для горизонтального масштабирования и для вертикального). Значения меньше единицы будут уменьшать размеры элемента, а значения больше единицы увеличивать (scale(1) не приведет к какому-нибудь результату).
div{
 transform: scale(0.5); // уменьшение размеров элемента в два раза
}
Функция rotate поворачивает элемент. Положительное значение поворачивает элемент по часовой стрелке, а отрицательное против часовой. Функция skew определяет наклон элемента. Принимает два числовых значения: наклон по оси X и по оси Y. Если передать только одно значение, наклон будет только по оси X.
div{
 transform: rotate(180grad) skew(10grad);
}

/* Поворот элемента на 180 градусов и небольшой его наклон */
Все эти функции преобразуют элемент относительно точки, которая изначально находиться в центре этого элемента, но с помощью свойства transform-origin эту точку можно сместить. Свойство принимает два числовых значения: смещение точки по оси X и по оси Y.
