СЛЕДУЮЩИЙ УРОК
К РАЗДЕЛАМ
Псевдоклассы определяют динамическое состояние элементов, которое изменяется с помощью действий пользователя (например изменение цвета блока при наведении мыши на него). При использовании псевдоклассов браузер не перегружает текущий документ, поэтому с помощью псевдоклассов можно получить разные динамические эффекты на странице. Синтаксис применения псевдоклассов такой:
Селектор:Псевдокласс{
 Описание правил стиля;
}

/* Селектор - элемент или группы элементов, к которым применяются стили */
В CSS существует более 30 псевдоклассов. Например, псевдоклассы link, active и visited определяют свойства ссылок в трех разных состояниях (ещё не посещенные, в момент клика и уже посещенные).
Псевдокласс cheked Применяется к элементам интерфейса, таким как переключатели (checkbox) и флажки (radio), когда они находятся в положение «включено».
Псевдоклассы disabled и enabled используются для определения стилей к заблокированым и не заблокированным элементам соответственно.
Псевдокласс hover определяет стиль элемента при наведении на него курсора мыши, но при этом по элементу еще не кликнули. Псевдокласс focus определяет стиль для элемента получающего фокус (элемент находится в фокусе, если по ниму кликнули мышкой).
Псевдокласс not задает правила стилей для элементов, которые не содержат указанный селектор. В качестве селектора могут указываться псевдоклассы (кроме самого псевдокласса not), теги, идентификаторы, классы и селекторы атрибутов.
input:not([type="submit"]){
 border: 1px solid #ccc;
}

/* Стиль применится ко всем input без атрибута type="submit" */
