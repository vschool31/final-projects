CSS - это каскадные таблицы стилей (Cascading Style Sheets). С их помощью осуществляется оформление вашей страницы. CSS позволяет настраивать цвета, смещать объекты, управлять переходами, создавать анимации и многое другое.
Есть три способа подключения CSS к вашей странице. Во-первых, вы можете прописывать стили отдельно в каждом теги. Для этого в теги нужно написать атрибут style и в его значении указывать свойства.
<div style = "background-color: black"></div> <!--Задали тегу div черный фон-->
Во-вторых, вы можете прописывать стили для всей страницы в самом её начале. В этом случае style нужно использовать как тег (свойства указываются между открывающим и закрывающим тегами), который нужно поместить между тегами <head></head>.
<head>
 <style>
  div{
   background-color: black; /* У всех div черный фон */
  }
 </style>
</head

/*Кстати, в CSS комментарии указываются таким способом*/
В-третьих можно создать отдельный документ, в котором будут находиться все стили (или несколько таких документов). В документ вы сразу записываете свойства.
div{
 background-color: black;
}
p{
 background-color: black;
}
Далее вы сохраняете документ с расширением .css (обычно этот файл называют style). Затем вам надо подключить этот документ к вашей HTML-странице. Для этого в теге <head> вы пишите строчку:
<link rel="stylesheet" type="text/css" href="путь к документу">
Из всех трех способов чаще всего используется третий (намного удобней, когда все ваши стили находятся в одном файле).
Особенность CSS в том, что вы можете менять не только все одинаковые теги, но и какие-то конкретно из них, так как в CSS имеются понятия классов и id. Указав класс или id, вы можете прописать для него отдельные стили.
 <!--HTML-->
<div class="some_class"></div>
<div id="some_id"></div>

 /*CSS*/
.some_class{
 background-color: black; /*Указание стилей класса (точка + имя класса)*/
}
#some_id{
 background-color: red; /*Указание стилей id (решетка + имя id)*/
}
Отличие между class и id состоит в том, что один и тот же class может быть у нескольких элементов, а вот id только у какого-то одного элемента.
В CSS вам понадобится указывать значения свойств и часто это будет какое-то численное значение. У таких значений есть две составляющие: число и единицы измерения. Например, запись "15px" означает 15 пикселей.
