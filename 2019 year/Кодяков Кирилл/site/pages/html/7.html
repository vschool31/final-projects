В HTML есть два вида списков: нумерованный и маркированный. Чтобы создать нумерованный список Вам потребуется тег <ol>, а для маркированного тег <ul>. Оба этих тега только создают контейнеры. Чтобы наполнить его записями используется тег <li> (один тег на одно положение в списке).
<ol>
 <li>Пункт 1</li>
 <li>Пункт 2</li>
 <li>Пункт 3</li>
</ol>

<ul>
 <li>Пункт 1</li>
 <li>Пункт 2</li>
 <li>Пункт 3</li>
</ul>
Оба вида списка можно настраивать с помощью атрибутов. У тега <ul> только одна настройка - вид маркера. Отвечает за это атрибут type: disc - кружок, circle - окружность, square - квадрат.
<ul type = "disc">
 <li>Пункт 1</li>
 <li>Пункт 2</li>
 <li>Пункт 3</li>
</ul>

<ul type = "circle">
 <li>Пункт 1</li>
 <li>Пункт 2</li>
 <li>Пункт 3</li>
</ul>

<ul type = "square">
 <li>Пункт 1</li>
 <li>Пункт 2</li>
 <li>Пункт 3</li>
</ul>
Тег <ol> немного богаче на настройки. Тем же атрибутом type можно настраивать вид маркера: A - заглавные латинские буквы, a - строчные латинские буквы, I - заглавные римские цифры, i - строчные римские цифры, 1 - арабские цифры. Атрибут reserved меняет нумерацию в списке на обратный порядок. Атрибут start указывает число, с которого будет начинаться список.
<ol type = "A">
 <li>Пункт 1</li>
 <li>Пункт 2</li>
 <li>Пункт 3</li>
</ol>

<ol type = "a">
 <li>Пункт 1</li>
 <li>Пункт 2</li>
 <li>Пункт 3</li>
</ol>

<ol type = "I">
 <li>Пункт 1</li>
 <li>Пункт 2</li>
 <li>Пункт 3</li>
</ol>

<ol type = "i">
 <li>Пункт 1</li>
 <li>Пункт 2</li>
 <li>Пункт 3</li>
</ol>

<ol type = "1">
 <li>Пункт 1</li>
 <li>Пункт 2</li>
 <li>Пункт 3</li>
</ol>

<ol reserved>
 <li>Пункт 1</li>
 <li>Пункт 2</li>
 <li>Пункт 3</li>
</ol>

<ol start = "4">
 <li>Пункт 1</li>
 <li>Пункт 2</li>
 <li>Пункт 3</li>
</ol>
