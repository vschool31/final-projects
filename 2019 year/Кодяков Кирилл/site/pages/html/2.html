Служебные теги используются для настройки вашего сайта для браузера, а также связывания файлов сайта. К служебным тегам относят <!--..-->, <!DOCTYPE>, <title>, <meta>, <script>, <style>, <link>
<!--..--> - тег комментария. Текст комментария не отображается на странице. Это удобно тем, что вы можете делать записи внутри своего кода (например, описать за что отвечает часть кода). В будущем, вернувшись к своему проекту, Вам не придется тратить время на его изучение, чтобы понять как он работает. В больших компаниях это особенно важно, так как с вашим кодом будут работать другие люди.
<!--Текст вашего комментария-->
Элемент <!DOCTYPE> предназначен для указания типа текущего документа — DTD (document type definition). Это необходимо, чтобы браузер понимал, как следует интерпретировать текущую веб-страницу (поскольку HTML существует в нескольких версиях). Кроме того, имеется XHTML (EXtensible HyperText Markup Language - расширенный язык разметки гипертекста), похожий на HTML, но различающийся с ним по синтаксису. Чтобы браузер «не путался» и понимал, согласно какому стандарту отображать веб-страницу и необходимо в первой строке кода задавать <!DOCTYPE>. Существует несколько видов <!DOCTYPE>, но в общем случае можно использовать самую простую конструкцию <!DOCTYPE html>, которая будет сообщать браузеру, что перед ним страница HTML.
<!DOCTYPE html>
Тег <title> определяет заголовок документа. Этот заголовок отображается в названиях вкладок вашего браузера.
<title> !Ваш заголовок! </title>
Тег <meta> выполняет множество различных функций: кодировка документа(charset), ключевые слова(keywords), описание(description) и авторство(author). Кодировка документа - означает установку набора символов для страницы. Каждый способ кодировки можно представить как табличку, в которой каждой букве русского алфавита, как строчной, так и заглавной, поставлен в соответствие некоторый код, которым эта буква представляется в компьютере.Если текст, записанный при помощи одной кодировки, попытаться прочитать при помощи другой, то вместо нормального текста отобразится беспорядочный набор символов. Ключевые слова - это отдельные слова и словосочетания, отражающие самую важную информацию, изложенную на странице, используемые поисковыми роботами. Описание отображается в поиске, под сайтом. Вся информация, записанная в этот тег, становится описанием вашего сайта, которое пользователь может прочитать, прежде чем зайти на сайт. Заполнив авторство, вы сообщаете, что данный сайт создали именно вы.
<meta charset = "UTF-8"> <!--Указание кодировки-->
<meta name = "keywords"content = "cюда вписываются ключевые слова"> <!--Указание ключевых слов-->
<meta name = "description" content = "cюда помещается описание web-страницы"> <!--Указание описания-->
<meta name = "author" content = "автор контента"> <!--Указание автора-->
Тег <script> описывает скрипты. Скрипт - это программа, предназначенная для распознавания и немедленного выполнения. Скрипты позволяют "оживлять" ваши страницы(например, анимация главной страницы выполнена с помощью скриптов). JavaScript - основной язык программирования, который используют для написания скриптов. Основные атрибуты этого тега: src, async, defer. src импортирует файл со скриптами из внешнего файла. Как только внешние инструкции полностью загружаются, браузер обрабатывает их, словно они были внедрены в текущий документ. Работать со скриптами намного удобней, когда они располагаются в отдельном файле. При наличии атрибута async браузер при возможности запускает скрипт асинхронно. Это означает, что указанный в атрибуте src файл будет выполняться без ожидания загрузки и отображения веб-страницы. В то же время и страница не ожидает результата выполнения скрипта, а продолжает загружаться как обычно. Атрибут defer откладывает выполнение скрипта до тех пор, пока вся страница не будет загружена полностью.
<script src = "cсылка на внешний файл"></script> <!--Подключение внешнего скрипта-->
<script async> ... </script> <!--Асинхронный скрипт-->
<script defer> ... </script> <!--Скрипт выполнится после загрузки страницы-->
Тег <style> применяется для определения стилей элементов веб-страницы. Указывается в теге <head>. Обычно все стили хранятся в отдельном файле, который привязывают к странице с помощью тега <link>. Более подробно стили и способы их указания рассмотрены в первом уроке по CSS.
