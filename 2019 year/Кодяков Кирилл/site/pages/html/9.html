Иногда вам потребуется вставить в ваш сайт какой-либо видео- или аудиоматериал. Для этого существую специальные теги. Начнём с видео.
Для вставки видео используют тег <video>. С помощью атрибута src задается ссылка на видео (или путь к видеофайлу).
<video src = "ссылка">></video>
Изначально у видео отсутствует панель управления. Чтобы её добавить используется атрибут controls.
<video src = "ссылка" controls = "controls"></video>
Также в видео можно добавить картинку, которая будет отображаться пока видео недоступно или не воспроизводится. Осуществляется это атрибутом poster со значение ссылки на картинку.
<video src = "ссылка" controls = "controls" poster = "ссылка"></video>
Для вставки аудиоконтента используется тег <audio>. Как и в случае с видео, для указания ссылки на аудиофайл используют атрибут src, а для добавления панели атрибут controls.
<audio src = "ссылка" controls = "controls"></audio>
