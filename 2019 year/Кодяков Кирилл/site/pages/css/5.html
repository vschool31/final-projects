В CSS огромное количество свойств отвечающих за текст. Свойство color отвечает за цвет текста. Значение сойства можно задавать несколькими способами: указать имя цвета, использовать шестнадцатеричные значения, использовать синтаксис rgb или hsl.
 /* Красный цвет, записанный разными способами*/
color: red;
color: #FF0000;
color: rgb(255, 0, 0);
color: hsl(0, 100%, 50%);
Свойство direction определяет направление написания текста. Значение ltr - слева направо, а значение rtl - справа налево.
Свойство letter-spacing устанавливает расстояние между символами в тексте. Свойство line-height устанавливает межстрочный интервал.
Свойство text-align указывает способ выравнивания содержимого по горизонтали (left - по левому краю, right - по правому краю, center - по центру, justify - по ширине).
Свойство text-decoration добавляет некоторые элементы декорирования к тексту. Значение underline рисует горизонтальную линию под текстом. Значение overline рисует горизонтальную под текстом. Значение line-through рисует линию, которая перечеркивает текст.
Свойство text-indent определяет отступ первой строки в тексте.
Свойство text-overflow определяет поведение текста, который не помещается в контейнер. Значение clip обрезает текст, не вошедший в контейнер. Значение ellipsis не только обрезает, но и отображает эллипсис (многоточие).
Свойство vertical-align определяет вертикальное выравнивание табличных элементов. Значение top выравнивает содержимое ячейки относительно её вверха. Значение middle выравнивает содержимое ячейки относительно её середины. Значение bottom выравнивает содержимое ячейки относительно её низа.
