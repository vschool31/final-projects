СЛЕДУЮЩИЙ УРОК
К РАЗДЕЛАМ
Зачастую нам надо повторять одно и то же действие во многих частях программы. Чтобы не повторять один и тот же код несколько раз были придуманы функции. Структура функций:
function название_функции(параметры){
 тело функции
}
Создается функция ключевым словом function, после которого идет название функции. По этому имени вызывают функцию.
function sum(){...} // создание функции
sum(); // вызов функции
Важной составляющей функции являются параметры. В круглых скобках можно указать какие параметры может принимать функция.
function sum(a, b){...}
В коде выше созданная функция sum может принимать параметры a и b. Эти параметры могут использоваться в теле функции.
function sum(a, b){
 alert(a+b); // вывести на экран сумму a и b
}
Теперь при вызове функции в круглые скобки указываются два числа. Их порядок соответствует порядку параметров (на место параметра a встанет первое число, на место параметра b встанет второе число).
sum(10, 12) // 22
В JavaScript есть свои собственные функции. Например, функция alert(), которая использовалась выше (выводит на экран содержимое скобок). Эти функции можно использовать без описания. Вместо alert() в данной функции можно было бы использовать директиву return. Данная директива показывает, что будет возвращать функция.
function sum(a, b){
 return a+b;
}
Например, запись alert(10 + sum(1,2)) будет выводить на экран число 13. В теме функций важным моментом являются локальные переменные. Локальные переменные - это переменные, объявленные внутри тела функции. Они используются только в этой функции и нигде больше.
