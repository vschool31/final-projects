К РАЗДЕЛАМ
Объекты и манипуляции с ними невозможно полностью описать в одном уроке. Здесь представлен небольшой пласт информации, который послужит основой в изучении объектов в языке JavaScript. Объект — структура данных, в которой можно хранить любые данные в формате ключ-значение. Для понимания можно представить шкаф с подписанными ящиками. В этих ящиках хранятся данные. По имени можно легко найти ящик и взять то значение, которое в нём лежит. Структура объекта:
var имя_объекта{
 ключ: значение,
 ключ: значение,
 ...
}
В отличии от массивов, в объектах можно хранить разные типы данных. Например:
var human = {
 name: "Витя",
 age: 45
}
Обращения к ключам (их еще называют свойствами) происходит через точку. Например, следующий код выведет значение свойства age:
alert(human.age)
К объекту можно добавлять новые свойства и удалять уже имеющиеся.
human.gender = "male"; //добавление нового свойства
delete human.gender; //удаление свойства
Помимо свойств у объектов могут методы. Методы - это функции, имеющиеся у объекта.
var calculator = {
 sum: function(a,b){
  return a+ b;
 }
}
Вызов методов аналогичен вызову свойств
alert(calculator.sum(1,2)); // 3
