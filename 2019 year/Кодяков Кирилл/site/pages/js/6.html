КОНСТРУКЦИЯ SWITCH
СЛЕДУЮЩИЙ УРОК
К РАЗДЕЛАМ
Конструкция switch заменяет собой сразу несколько if. Она представляет собой более наглядный способ сравнить выражение сразу с несколькими вариантами. Синтаксис таков:
switch(переменная) {
 case значение:
  ...
 break;

 case значение:
  ...
 break;

 default:
  ...
 break;
}
После switch указывается переменная, которую мы будем сравнивать. После case указывается проверяемое значение. Указанная нами переменная строго сравнивается с этим значением. Если они равны, выполняется дальнейший код до break (дальнийшие проверки не осуществляются). Если ни одна из проверок не прошла, выполняется код после default (если он указан). Пример программы:
var a = 3;
var b;

switch(a) {
 case 2: // if(a===2){...}
  b = a;
 break;

 case 3: // if(a===3){...}
  b = a; //выполнится это выражение
 break;
}
