ЦИКЛЫ
СЛЕДУЮЩИЙ УРОК
К РАЗДЕЛАМ
Зачастую в программе какое-то действие нужно сделать несколько раз. Для многократного повторения одного участка кода – предусмотрены циклы. Их существует три вида. Первый из них - цикл for. Его структура:
for(начало; условие; шаг){
 тело цикла
}
В "начало" помещается переменная. Оно будет выполнять роль счётчика. Затем идет условие. Пока это условие истинно, будет выполняться тело цикла. В блоке "шаг" определяются изменения, происходящие с переменной-счетчиком после каждой итерации (итерация - повторение тела цикла).
for(var i = 0; i < 3; i++){...}
В данном примере тело цикла выполнится 3 раза. Почему? "Начало" выполняется один единственный раз при входе в цикл (var i = 0). Затем проверяется условие (i < 3), и, если оно истинно, выполняется тело цикла. После этого выполняется "шаг" (i++). Затем опять проверяется условие. Как только условие окажется ложным, цикл перестанет выполняться. Второй вид цикла - цикл while. Его структура:
while (условие) {
 тело цикла
}
В отличии от цикла for, в этом цикле нет счётчика, а значит нам заранее не известно количество итераций. В этом цикле сначала проверяется условие, и если оно иситинно, то выполняется тело цикла. Третий тип цикла - do...while. Его структура:
do {
 тело цикла
} while (условие)
Отличие этого цикла от обычного while заключатся в том, что тело цикла выполнится минимум один раз, потому что условие проверяется после итерации. Также в циклах используют две директивы: break и continue. Как только интерпретатор встречает в цикле директиву break, он выходит из цикла.
while (i < 0) {
 if(i == -3){break}; //если i равно -3, выйти из цикла
 ...
}
Директива continue похожа на директиву break, но вместо выхода из цикла она просто пропускает текущую итерацию.
while (i < 0) {
 if(i == -3){continue}; //если i равно -3, пропустить итерацию
 ...
}
